	
--- Adresár \database --- 

adresár \database\sql -
Obsahuje súbory SQL pre vytvorenie tabuliek databázy - \create_tables 
a súbory pre vytvorenie náhľadov tabuliek - \create_views
Súbory SQL je potrebné importovať do databázy - názov vlastnej databázy je potrebné nastaviť v \src\php_src\database.php

adresár \database\upload - miesto pre ukladanie príloh odoslaných do databázy 


--- Adresár \src --- 

Adresár obsahuje podadresáre:

\src\css - kaskádové štýly css
\src\js_src - zdrojové súbory javascript - \script.js
	    - ostatné - súbroy pre vytvorenie nového fontu
\src\php_src - \database.php - vytvorenie spojenia s databázou, nastavenie servera, používateľa, hesla a mena databázy
	     - ostatné - odosielanie údajov do databázy


--- Adresár \web --- 

adresár obsahuje statické komponenty - HTML stránky
\images - obsahuje obrázky aplikácie



Štart HTML aplikácie: localhost/erasmus/web/erasmus.html
